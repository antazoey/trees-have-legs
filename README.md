# Pharcobial

A game (WIP).

## Installation

To install the game, make sure you have `poetry` installed and run:

```bash
poetry install
```

## Quick Play

To play, run:

```bash
play_pharcobial
```

To play without monsters chasing you, run:

```bash
play_pharcobial --num-monsters 0
```

For more options, run:

```bash
play_pharcobial --help
```
